YouTube Transcript Summarizer Web Extension Using NLP

In this project, I have developed a systematic solution for summarizing the text of a YouTube video. I discovered the simplest way to achieve this summarization task. Regarding the performance of the summarization, I utilized a Hugging Face transformer for text summarization. This choice was made because it boasts a pre-trained model and is widely employed for various Natural Language Processing (NLP) tasks. • To streamline the process of obtaining a summary for a YouTube video and minimize complexity, I created a Chrome extension. Users can simply add the extension to their browser and click on the "Summarize" button for a particular YouTube video.
